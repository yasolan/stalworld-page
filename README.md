# STALWORLD Page

Bug tracker for [stalworld.ru](https://stalworld.ru/) — hosted on GitHub Pages.

**Live:** https://yasolan.github.io/stalworld-page/

## Pages

- `/` — bug list
- `/report.html` — submit bug (GitHub Issue)
- `/admin.html` — edit bugs (password in `js/admin.js`)

## Update bugs

Edit `data/bugs.json` or use admin panel → download JSON → commit & push.

## Local preview

```powershell
python -m http.server 8080
```
